# Практична робота "Створення і використання статичних методів"

## 📋 Опис

Цей проєкт демонструє вирішення практичного завдання із застосуванням **статичних методів** у мові Java.  
У програмі реалізовано пошук **простих чисел** у межах заданого числового діапазону.  
Метод `Calculate` повертає результат у вигляді **рядка**, де всі прості числа відокремлені комами.
